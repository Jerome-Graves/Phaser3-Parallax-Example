# Phaser 3 Parallax Example

A simple scrolling parallax scene built with [Phaser 3](https://phaser.io/) and Webpack.
A minimal, readable starting point for layered 2D backgrounds.

![Parallax scrolling demo](example-gif.gif)

Live demo: <https://codesandbox.io/s/phaser3-parallax-example-rdv72>

## Run

```bash
npm install
npm start         # webpack-dev-server, opens the browser
npm run build     # production build
```
